## 🚀 Projeto em destaque

**🛰️ OASIS — Orbital Awareness & Safety Intelligence System**

Projeto **vencedor da Global Solution 2026.1 (FIAP)**. Sistema de IA preventiva que classifica satélites ativos e detritos espaciais a partir de dados orbitais reais do CelesTrak, alcançando **99,6% de acurácia** na detecção de detritos com Machine Learning (KNN). O projeto integra, de ponta a ponta:

- **Machine Learning:** classificador KNN em scikit-learn sobre ~18.200 objetos reais
- **Rede Neural:** MLPClassifier (ReLU + Adam) para classificação
- **Análise Estatística:** EDA e estatística descritiva das variáveis orbitais
- **Banco de Dados:** modelagem relacional em MySQL (3FN) com procedure e function
- **Aplicação Python:** cálculo de risco de colisão e geração de alertas
- **Cibersegurança:** OSINT e integridade de dados via hash SHA-256
- **IoT:** monitoramento ambiental com ESP32 e sensores

➡️ [Ver repositório](https://github.com/brunoferreira-analytics/oasis-orbital-safety)
